- BGP c'est un protocole (Un outil).
- BGP EVPN utilise le protocol BGP. Il permet au routeur de dire qu'elles ordinateurs est connecté avec eux. 
- RR ( Route Reflector) c'est un routeur qui utilise le protocole BGP pour faire circuler les informations entre les routeurs, pas les données.

OSPF C'est un autre protocole. Il sert a relier les routeurs entre eux.
