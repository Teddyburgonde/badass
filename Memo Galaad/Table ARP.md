table ARP: la table ARP (Address Resolution Protocol) est un petit dictionnaire (ou cache) stocké dans la mémoire de chaque ordinateur ou routeur.

    Contenu : Elle associe une adresse IP (ex: 192.168.1.10) à une adresse MAC (ex: 52:43:13:09:8F:99).

Utilité : Elle évite de devoir demander "Qui est qui ?" à chaque fois qu'on envoie un message. L'ordinateur regarde d'abord sa table ; s'il ne connaît pas la MAC, il lance une requête ARP.
